Features:
## شرح المشروع
نظام إدارة المخزون بنسخة الكونسول هو تطبيق بايثون بسيط لإدارة المنتجات والمخزون باستخدام قاعدة بيانات SQLite. يتيح للمستخدم إضافة المنتجات، عرضها، تعديلها، حذفها، وتصدير تقرير المنتجات إلى ملف CSV. يعتمد على البرمجة الكائنية (OOP) لسهولة التطوير والصيانة.

## المميزات
- إضافة وتعديل وحذف المنتجات
- عرض جميع المنتجات
- تصدير تقرير المنتجات إلى ملف CSV
- هيكلة احترافية وفصل الأكواد

## طريقة التشغيل
1. افتح المجلد في VS Code
2. شغّل البرنامج عبر الأمر:
	```
	python -m app.main
	```
3. اتبع التعليمات في القائمة لإدارة المنتجات.

Technologies:
- Python
- SQLite
