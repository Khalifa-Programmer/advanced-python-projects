Features:
## شرح المشروع
أداة بايثون لأتمتة وتنظيم الملفات داخل المجلدات بشكل تلقائي حسب نوع الامتداد. تساعد في ترتيب الملفات وتقليل العمل اليدوي المتكرر، وتناسب الاستخدام المكتبي والشخصي.

## المميزات
- تنظيم الملفات حسب الامتداد
- نقل الملفات تلقائيًا إلى مجلدات مخصصة
- هيكلة احترافية وسهولة التشغيل

## طريقة التشغيل
1. افتح المجلد في VS Code
2. شغّل البرنامج:
	```
	python -m app.main
	```
3. أدخل مسار المجلد المطلوب تنظيمه عند التشغيل.
