## شرح المشروع
نظام إدارة المخزون بواجهة سطح المكتب هو تطبيق احترافي باستخدام بايثون وTkinter لإدارة المنتجات والمخزون عبر واجهة رسومية عصرية. يتيح إضافة وتعديل وحذف المنتجات، عرضها في جدول منسق، وتصدير تقرير إلى CSV. مناسب للاستخدام المكتبي وسهل التشغيل.

## المميزات
- واجهة رسومية حديثة وسهلة الاستخدام
- إضافة وتعديل وحذف المنتجات
- عرض المنتجات في جدول احترافي
- تصدير تقرير المنتجات إلى ملف CSV

## طريقة التشغيل
1. افتح المجلد في VS Code
2. شغّل البرنامج عبر الأمر:
	```
	python -m app.main
	```
3. ستظهر نافذة البرنامج لإدارة المنتجات.

Technologies:
- Python
- Tkinter
- SQLite
