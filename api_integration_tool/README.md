## شرح المشروع
أداة بايثون لجلب ومعالجة البيانات من واجهات REST API وتخزينها في قاعدة بيانات MySQL. تشمل معالجة البيانات، التعامل مع الأخطاء، وتسجيل الأحداث في ملف لوج. مناسبة لدمج البيانات الخارجية في قواعد بيانات محلية.

## المميزات
- جلب بيانات المستخدمين من API عام
- تخزين البيانات في MySQL
- معالجة الأخطاء وتسجيلها
- هيكلة احترافية وفصل الأكواد

## طريقة التشغيل
1. افتح المجلد في VS Code
2. ثبت المتطلبات:
	```
	pip install -r requirements.txt
	```
3. تأكد من إعداد قاعدة بيانات MySQL باسم api_data
4. شغّل البرنامج:
	```
	python -m app.main
	```

Technologies:
- Python
- requests
- mysql-connector-python
