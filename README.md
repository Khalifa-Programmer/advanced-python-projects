# Advanced Python Projects

هذا المستودع يحتوي على مجموعة مشاريع أكاديمية متقدمة بلغة بايثون، كل مشروع مستقل بهيكلة احترافية وملف README خاص به.

## المشاريع المتوفرة:

### 1. Inventory Management System (Console)
- نظام إدارة مخزون عبر الكونسول باستخدام Python وSQLite.
- يدعم CRUD وتصدير CSV.
- [تفاصيل المشروع](inventory_system/README.md)

### 2. Inventory Desktop System (Tkinter)
- نظام إدارة مخزون بواجهة سطح مكتب عصرية باستخدام Tkinter.
- يدعم CRUD، جدول احترافي، وتصدير CSV.
- [تفاصيل المشروع](inventory_desktop/README.md)

### 3. API Integration Tool
- أداة لجلب ومعالجة البيانات من REST API وتخزينها في MySQL.
- يدعم معالجة الأخطاء وتسجيل الأحداث.
- [تفاصيل المشروع](api_integration_tool/README.md)

### 4. File Automation Toolkit
- أداة لتنظيم الملفات تلقائيًا حسب الامتداد.
- تسهل ترتيب الملفات وتقليل العمل اليدوي.
- [تفاصيل المشروع](file_automation_toolkit/README.md)
